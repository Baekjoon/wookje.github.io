[1019번: 책 페이지](https://www.acmicpc.net/problem/1019)

## 풀이

나중에 써야지~

## 코드

```cpp
#include <stdio.h>

int N, prev;
int cnt[10];
int main() {
	scanf("%d", &N);

	for (int i = 1; N; i *= 10) {
		int now = N % 10; N /= 10;
		for (int j = 0; j < now; ++j) cnt[j] += (N + 1) * i;
		cnt[now] += N * i + prev + 1;
		for (int j = now + 1; j < 10; ++j) cnt[j] += N * i;
		cnt[0] -= i;
		prev += now * i;
	}
	for (int i = 0; i < 10; ++i) printf("%d ", cnt[i]);
}
```
