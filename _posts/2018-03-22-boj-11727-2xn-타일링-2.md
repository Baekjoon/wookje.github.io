## 코드

```cpp
main() {
    int n, a = 0, b = 1, i;
    scanf("%d", &n);
    for (i = 0; i < n; ++i) {
        b += a * 2;
        a = b - a * 2;
        b %= 10007;
    }
    printf("%d", b);
}
```
